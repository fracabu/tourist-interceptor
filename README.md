<a name="english"></a>
## :gb: English

### Overview

A CRM for B2B lead generation and outreach in Rome's events and congress sector, specifically for vacation rental businesses.

### Features

- **Lead Management** - Track business contacts
- **Events Focus** - Congress and events sector
- **Outreach Tools** - Email and contact campaigns
- **Rome Market** - Specialized for Rome area
- **Vacation Rentals** - Target audience focus

### Quick Start

```bash
git clone https://github.com/fracabu/tourist-interceptor.git
cd tourist-interceptor
# Open index.html in browser
```
